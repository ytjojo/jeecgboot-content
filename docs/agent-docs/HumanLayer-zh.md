## 让 Coding Agents 在复杂代码库中解决困难问题的最佳方式

**CodeLayer 是一个开源 IDE，用于编排 AI coding agents。**

它内置了经实战验证的工作流，让 AI 能在大型、复杂的代码库里解决棘手问题。

基于 Claude Code 构建。开源。从你的笔记本扩展到整个团队。

## 来自打造 “Context Engineering” 的团队

在如何最大化发挥当今模型能力方面的顶级专家。

这场在 YC（2025 年 8 月 20 日）分享的演讲，奠定了使用 AI 在复杂代码库中解决困难问题的基础方法论。
