### 漂移3: API POST请求写法不规范 ⚠️ 中（代码规范）

**实际实现**: 5个POST接口使用`params`传body
- changeUserStatus: line 44
- releaseUser: line 48
- batchReleaseUsers: line 52
- verifySecurity: line 79
- sendVerifyCode: line 83

**漂移影响**:
- 项目拦截器有兼容逻辑（非GET请求自动将params转data），功能可工作
- 写法不规范，易引起误解
- 与其他模块代码风格不一致

**修复建议**: 将`params: payload`改为`data: payload`
