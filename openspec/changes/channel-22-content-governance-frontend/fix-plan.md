**来源**: drift-report-20260627-084036.md, backend-issues.md
**位置**: src/api/content/channel/ 所有API文件
**优先级**: BLOCK
**依赖**: BE-008（后端API路径重构完成）
**类型**: 代码修复-前端

**修复步骤**:
1. 待后端完成API路径重构（BE-008）后，同步更新所有前端API调用路径：
   - 旧路径：`/api/v1/content/channel/publish/*` → 新路径：`/api/v1/content/channels/{channelId}/publications/*`
   - 旧路径：`/api/v1/content/channel/review/*` → 新路径：`/api/v1/content/channels/{channelId}/publications/reviews/*`

3. 同步更新HTTP方法：查询接口用GET，写操作用POST
4. 注意：前端API封装不需要写`/api/v1/`前缀（由baseURL统一处理），实际路径如`/content/channels/{channelId}/...`

**验证方式**:

- 联调时无404错误
- 所有组件单元测试更新后通过
