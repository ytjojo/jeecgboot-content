## Why

内容社区当前缺少独立的认证前端体系，注册登录依赖平台层，无设备管理、账号安全设置、风控交互和注销流程的前端入口。需要为内容社区构建完整的用户认证与账号安全前端能力，覆盖注册、登录、会话管理、账号绑定、密码找回、风控拦截和账号注销全流程，目标是注册转化率提升 25%、登录成功率 >99.5%。

## What Changes

- 新增注册页面：支持手机号验证码注册（含图形验证码防刷）和邮箱密码注册（含邮箱验证确认流程）
- 新增登录页面：支持密码登录（手机号/邮箱）和验证码登录（手机号 + 短信验证码），以及第三方登录（微信/Apple/Google）
- 新增多端会话管理页面：设备列表查看、主动下线、信任设备管理
- 新增账号安全页面：手机号/邮箱绑定与换绑解绑、第三方账号绑定与解绑
- 新增密码找回页面：手机号/邮箱重置密码，三步流程（验证身份 → 设置新密码 → 完成）
- 新增异常登录通知弹窗与消息中心通知列表
- 新增风控拦截交互组件：图形验证码弹窗、账号锁定提示
- 新增账号注销页面：注销前检查、确认弹窗、冷静期状态页
- 扩展 `useUserStore`：新增登录方式、账号安全状态、设备数、注销状态、异常通知数等状态
- 新增注册后个性化推荐引导弹窗（兴趣标签选择）
- 新增前端埋点事件体系：注册漏斗、登录漏斗、账号安全事件

## Capabilities

### New Capabilities

- `user-registration`: 用户注册能力，包含手机号注册、邮箱注册、注册后引导完善资料和个性化推荐
- `user-login`: 用户登录能力，包含密码登录、验证码登录、第三方登录、来源页跳转
- `session-management`: 多端会话管理能力，包含设备列表、设备下线、信任设备管理
- `account-binding`: 账号绑定与解绑能力，包含手机号/邮箱绑定换绑解绑、第三方账号绑定解绑
- `password-recovery`: 密码找回能力，包含手机号/邮箱重置密码、三步流程
- `anomaly-notification`: 异常登录通知能力，包含实时弹窗通知、消息中心历史记录、确认/否认操作
- `risk-control`: 风控拦截交互能力，包含图形验证码、账号锁定提示、冷却倒计时
- `account-cancellation`: 账号注销能力，包含注销前检查、冷静期管理、取消注销
- `auth-analytics`: 前端埋点能力，包含注册漏斗、登录漏斗、账号安全事件上报

## Impact

- **新增页面文件**: 登录页、注册页、账号安全页、设备管理页、密码找回页、账号注销页
- **新增组件**: 异常登录通知弹窗、风控验证弹窗、密码强度指示器、个性化推荐引导弹窗
- **Store 扩展**: `useUserStore` 新增认证相关状态字段
- **API 层**: 新增约 30 个 API 接口封装（认证、账号安全、注销、风控验证）
- **路由配置**: 新增登录/注册/账号安全相关路由
- **依赖**: Ant Design Vue 4（已集成）、现有 `defHttp` 工具、`useMessage` / `useModal` hooks
- **响应式**: 全部页面需适配 PC / 平板 / 移动端三种布局
