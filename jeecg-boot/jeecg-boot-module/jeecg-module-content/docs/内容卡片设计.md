# 内容卡片设计方案

## 1. 概述

### 1.1 设计目标
基于现有的投票卡片、视频卡片、音频卡片基础上，扩展更多卡片类型，包括系统支持内容(文章、帖子、笔记、问答问题、问答回答、视频)卡片、链接卡片、文档卡片、商品卡片、活动卡片、位置卡片等，构建一个完整的卡片生态系统。

### 1.2 设计原则
- **统一性**：所有卡片遵循统一的JSON格式规范
- **扩展性**：支持新卡片类型的快速扩展
- **复用性**：卡片可以在不同内容类型中复用
- **交互性**：支持用户与卡片的交互操作
- **数据完整性**：确保卡片数据的完整性和一致性

## 2. 现有卡片类型分析

### 2.1 投票卡片 (poll)
**功能特点**：
- 支持单选/多选投票
- 实时统计投票结果
- 支持投票截止时间
- 用户可查看投票历史

**JSON格式**：
```json
{

  "data": {
    "cardType": "poll",
    "id": 601,
    "title": "你最喜欢的编程语言是？",
```

## 4. 数据库设计

### 4.1 卡片主表 (content_cards)

```sql
CREATE TABLE content_cards (
    id BIGSERIAL PRIMARY KEY,
    card_type VARCHAR(50) NOT NULL COMMENT '卡片类型：poll, video_card, audio_card, content_card, link_card, document_card, product_card, event_card, location_card, ad_card',
    title VARCHAR(500) NOT NULL COMMENT '卡片标题',
    description TEXT COMMENT '卡片描述',
    card_data JSONB NOT NULL COMMENT '卡片具体数据，存储JSON格式',
    creator_id BIGINT NOT NULL COMMENT '创建者ID',
    status SMALLINT DEFAULT 1 COMMENT '状态：0-禁用，1-启用，2-审核中',
    view_count BIGINT DEFAULT 0 COMMENT '查看次数',
    interaction_count BIGINT DEFAULT 0 COMMENT '交互次数（点击、投票等）',
    tags VARCHAR(1000) COMMENT '标签，逗号分隔',
    created_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    deleted_at TIMESTAMP NULL COMMENT '删除时间（软删除）',
    
    INDEX idx_card_type (card_type),
    INDEX idx_creator_id (creator_id),
    INDEX idx_status (status),
    INDEX idx_created_time (created_time),
    INDEX idx_deleted_at (deleted_at)
) COMMENT='内容卡片主表';
```

### 4.2 卡片交互记录表 (card_interactions)

```sql
CREATE TABLE card_interactions (
    id BIGSERIAL PRIMARY KEY,
    card_id BIGINT NOT NULL COMMENT '卡片ID',
    user_id BIGINT NOT NULL COMMENT '用户ID',
    interaction_type VARCHAR(50) NOT NULL COMMENT '交互类型：view, click, vote, like, share, download, register',
    interaction_data JSONB COMMENT '交互数据，如投票选项、点击位置等',
    ip_address VARCHAR(45) COMMENT 'IP地址',
    user_agent TEXT COMMENT '用户代理',
    created_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '交互时间',
    
    INDEX idx_card_id (card_id),
    INDEX idx_user_id (user_id),
    INDEX idx_interaction_type (interaction_type),
    INDEX idx_created_time (created_time),
    UNIQUE KEY uk_card_user_type (card_id, user_id, interaction_type)
) COMMENT='卡片交互记录表';
```

### 4.3 投票选项表 (poll_options)

```sql
CREATE TABLE poll_options (
    id BIGSERIAL PRIMARY KEY,
    card_id BIGINT NOT NULL COMMENT '投票卡片ID',
    option_text VARCHAR(500) NOT NULL COMMENT '选项文本',
    option_order SMALLINT NOT NULL COMMENT '选项顺序',
    vote_count BIGINT DEFAULT 0 COMMENT '投票数',
    created_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    
    INDEX idx_card_id (card_id),
    INDEX idx_option_order (option_order)
) COMMENT='投票选项表';
```

### 4.4 投票记录表 (poll_votes)

```sql
CREATE TABLE poll_votes (
    id BIGSERIAL PRIMARY KEY,
    card_id BIGINT NOT NULL COMMENT '投票卡片ID',
    option_id BIGINT NOT NULL COMMENT '选项ID',
    user_id BIGINT NOT NULL COMMENT '投票用户ID',
    ip_address VARCHAR(45) COMMENT 'IP地址',
    created_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '投票时间',
    
    INDEX idx_card_id (card_id),
    INDEX idx_option_id (option_id),
    INDEX idx_user_id (user_id),
    INDEX idx_created_time (created_time),
    UNIQUE KEY uk_card_user (card_id, user_id)
) COMMENT='投票记录表';
```

### 4.5 活动报名表 (event_registrations)

```sql
CREATE TABLE event_registrations (
    id BIGSERIAL PRIMARY KEY,

    created_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '报名时间',
    updated_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    
    INDEX idx_card_id (card_id),
    INDEX idx_user_id (user_id),
    INDEX idx_payment_status (payment_status),
    INDEX idx_created_time (created_time),
    UNIQUE KEY uk_card_user (card_id, user_id)
```

### 4.6 广告统计表 (ad_statistics)

```sql
CREATE TABLE ad_statistics (
    id BIGSERIAL PRIMARY KEY,
    card_id BIGINT NOT NULL COMMENT '广告卡片ID',
    stat_date DATE NOT NULL COMMENT '统计日期',
    impressions BIGINT DEFAULT 0 COMMENT '展示次数',
    clicks BIGINT DEFAULT 0 COMMENT '点击次数',
    conversions BIGINT DEFAULT 0 COMMENT '转化次数',
    spend DECIMAL(10,2) DEFAULT 0 COMMENT '花费金额',
    revenue DECIMAL(10,2) DEFAULT 0 COMMENT '收入金额',
    created_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    
    INDEX idx_card_id (card_id),
    INDEX idx_stat_date (stat_date),
    UNIQUE KEY uk_card_date (card_id, stat_date)
) COMMENT='广告统计表';
```

### 4.7 卡片标签关联表 (card_tags)

```sql
CREATE TABLE card_tags (
    id BIGSERIAL PRIMARY KEY,
    card_id BIGINT NOT NULL COMMENT '卡片ID',
    tag_id BIGINT NOT NULL COMMENT '标签ID',
    created_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    
    INDEX idx_card_id (card_id),
    INDEX idx_tag_id (tag_id),
    UNIQUE KEY uk_card_tag (card_id, tag_id)
) COMMENT='卡片标签关联表';
```

## 5. 卡片管理API设计

### 5.1 卡片CRUD操作

#### 5.1.1 创建卡片
```http
POST /api/cards
Content-Type: application/json

{
  "type": "card",
  "data": {
    "cardType": "article_card",
    "title": "深入理解Vue 3响应式原理",
    "description": "本文将深入探讨Vue 3响应式系统的实现原理",
    "cardData": {

    },
    "tags": ["Vue3", "响应式", "前端框架"]
  }
}
```

#### 5.1.2 获取卡片详情
```http
GET /api/cards/{cardId}
```

#### 5.1.3 更新卡片
```http
PUT /api/cards/{cardId}
Content-Type: application/json

{
```

```vue
<template>
```

```vue
import VideoCardEditor from './editors/VideoCardEditor.vue'
import ArticleCardEditor from './editors/ArticleCardEditor.vue'

    PollEditor,
    VideoCardEditor,
    ArticleCardEditor

  },

      selectedCardType: 'poll',
      cardData: {},

        { label: '投票卡片', value: 'poll' },
        { label: '视频卡片', value: 'video_card' },
        { label: '音频卡片', value: 'audio_card' },
        { label: '文章卡片', value: 'article_card' },
        { label: '链接卡片', value: 'link_card' },
        { label: '文档卡片', value: 'document_card' },
        { label: '商品卡片', value: 'product_card' },
        { label: '活动卡片', value: 'event_card' },
        { label: '位置卡片', value: 'location_card' },
        { label: '广告卡片', value: 'ad_card' }

        'poll': 'PollEditor',
        'video_card': 'VideoCardEditor',
```
