# 草稿自动保存快照设计方案

## 1. 概述

本文档详细描述了内容社区系统中草稿自动保存功能的快照设计方案，旨在为用户提供可靠的数据保护和版本管理能力。

## 2. 设计目标

- **数据安全**：防止用户编辑内容丢失
- **性能优化**：最小化存储开销和保存延迟
- **版本管理**：支持历史版本查看和恢复
- **智能保存**：根据用户行为智能触发保存
- **存储优化**：通过增量保存和压缩减少存储空间

## 3. 当前表结构分析

### 3.1 现有设计

```sql
CREATE TABLE IF NOT EXISTS draft_auto_saves (
    id VARCHAR(32) NOT NULL PRIMARY KEY COMMENT '自动保存ID',
    draft_id VARCHAR(32) NOT NULL COMMENT '草稿ID',
    save_data JSONB NOT NULL COMMENT '保存的数据快照',
    save_type INTEGER DEFAULT 1 COMMENT '保存类型：1-自动保存 2-手动保存',

    data_size INTEGER COMMENT '数据大小(字节)',

);
```

### 3.2 设计优势

- 使用JSONB存储，支持复杂数据结构
- 包含性能监控字段
- 区分自动保存和手动保存

### 3.3 设计不足

- 缺少版本控制机制
- 没有增量保存支持
- 缺少过期管理策略
- 没有内容哈希校验

## 5. 快照策略设计

### 5.1 分层快照策略

#### L1快照（实时级 - 5秒间隔）
- **触发条件**：内容变更、光标移动
- **保存内容**：仅关键字段变更
- **保留策略**：最近10个快照
- **存储方式**：增量快照

#### L2快照（分钟级 - 2分钟间隔）
- **触发条件**：定时器、重要内容变更
- **保存内容**：完整内容快照
- **保留策略**：最近50个快照
- **存储方式**：全量快照

#### L3快照（里程碑级）
- **触发条件**：用户手动保存、发布前保存
- **保存内容**：完整内容+元数据
- **保留策略**：永久保存
- **存储方式**：全量快照

### 5.2 智能触发机制

```java
/**
 * 智能保存触发器
 * 根据用户行为和内容变更智能决定保存时机
 */
@Component
public class SmartSaveTrigger {
    
    private static final int MIN_SAVE_INTERVAL = 5000; // 最小保存间隔5秒
    private static final int MAX_SAVE_INTERVAL = 120000; // 最大保存间隔2分钟
    private static final int SIGNIFICANT_CHANGE_THRESHOLD = 50; // 重要变更阈值
    
    /**
     * 判断是否需要触发保存
     * @param lastSaveTime 上次保存时间
     * @param changeCount 变更次数
     * @param contentLength 内容长度
     * @return 是否需要保存
     */
    public boolean shouldTriggerSave(long lastSaveTime, int changeCount, int contentLength) {
        long timeSinceLastSave = System.currentTimeMillis() - lastSaveTime;
        
        // 最小间隔保护
        if (timeSinceLastSave < MIN_SAVE_INTERVAL) {
            return false;
        }
        
        // 最大间隔强制保存
        if (timeSinceLastSave > MAX_SAVE_INTERVAL) {
            return true;
        }
        
        // 重要变更触发保存
        if (changeCount > SIGNIFICANT_CHANGE_THRESHOLD) {
            return true;
        }
        
        // 根据内容长度动态调整保存频率
        int dynamicInterval = Math.min(MAX_SAVE_INTERVAL, 
            Math.max(MIN_SAVE_INTERVAL, contentLength / 10));
        
        return timeSinceLastSave > dynamicInterval;
    }
}
```

## 6. 核心算法实现

### 6.1 增量快照计算

```java
/**
 * 增量快照计算器
 * 计算两个版本之间的差异，生成增量快照
 */
@Service
public class IncrementalSnapshotCalculator {
    
    /**
     * 计算增量快照
     * @param currentContent 当前内容
     * @param previousSnapshot 上一个快照
     * @return 增量快照数据
     */
    public JSONObject calculateIncremental(JSONObject currentContent, JSONObject previousSnapshot) {
        JSONObject incrementalData = new JSONObject();
        incrementalData.put("version", "1.0");
        incrementalData.put("snapshot_type", "incremental");
        incrementalData.put("timestamp", Instant.now().toString());
        incrementalData.put("parent_snapshot_id", previousSnapshot.getString("id"));
        
        JSONArray operations = new JSONArray();
        
        // 比较并生成操作列表
        compareAndGenerateOperations(currentContent, previousSnapshot.getJSONObject("content"), "", operations);
        
        incrementalData.put("changes", new JSONObject().fluentPut("operations", operations));
        
        return incrementalData;
    }
    
    /**
     * 递归比较对象并生成操作
     */
    private void compareAndGenerateOperations(Object current, Object previous, String path, JSONArray operations) {
        if (current == null && previous == null) {
            return;
        }
        
        if (current == null) {
            // 删除操作
            operations.add(createOperation("delete", path, previous, null));
            return;
        }
        
        if (previous == null) {
            // 新增操作
            operations.add(createOperation("insert", path, null, current));
            return;
        }
        
        if (!Objects.equals(current, previous)) {
            if (current instanceof JSONObject && previous instanceof JSONObject) {
                // 递归比较对象
                JSONObject currentObj = (JSONObject) current;
                JSONObject previousObj = (JSONObject) previous;
                
                Set<String> allKeys = new HashSet<>();
                allKeys.addAll(currentObj.keySet());
                allKeys.addAll(previousObj.keySet());
                
                for (String key : allKeys) {
                    String newPath = path.isEmpty() ? key : path + "." + key;
                    compareAndGenerateOperations(
                        currentObj.get(key), 
                        previousObj.get(key), 
                        newPath, 
                        operations
                    );
                }
            } else {
                // 更新操作
                operations.add(createOperation("update", path, previous, current));
            }
        }
    }
    
    /**
     * 创建操作对象
     */
    private JSONObject createOperation(String type, String path, Object oldValue, Object newValue) {
        JSONObject operation = new JSONObject();
        operation.put("type", type);
        operation.put("path", path);
        if (oldValue != null) {
            operation.put("old_value", oldValue);
        }
        if (newValue != null) {
            operation.put("new_value", newValue);
        }
        return operation;
    }
}
```

### 6.2 快照重建算法

```java
/**
 * 快照重建服务
 * 从增量快照链重建完整内容
 */
@Service
public class SnapshotRebuilder {
    
    @Autowired
    private DraftAutoSaveMapper autoSaveMapper;
    
    /**
     * 重建指定快照的完整内容
     * @param snapshotId 目标快照ID
     * @return 重建的完整内容
     */
    public JSONObject rebuildContent(String snapshotId) {
        // 1. 获取目标快照
        DraftAutoSave targetSnapshot = autoSaveMapper.selectById(snapshotId);
        if (targetSnapshot == null) {
            throw new BusinessException("快照不存在");
        }
        
        JSONObject snapshotData = JSONObject.parseObject(targetSnapshot.getSaveData());
        
        // 2. 如果是全量快照，直接返回
        if ("full".equals(snapshotData.getString("snapshot_type"))) {
            return snapshotData.getJSONObject("content");
        }
        
        // 3. 如果是增量快照，需要重建
        return rebuildFromIncremental(targetSnapshot);
    }
    
    /**
     * 从增量快照重建内容
     */
    private JSONObject rebuildFromIncremental(DraftAutoSave incrementalSnapshot) {
        // 1. 找到快照链
        List<DraftAutoSave> snapshotChain = buildSnapshotChain(incrementalSnapshot);
        
        // 2. 找到基础全量快照
        JSONObject baseContent = null;
        for (DraftAutoSave snapshot : snapshotChain) {
            JSONObject data = JSONObject.parseObject(snapshot.getSaveData());
            if ("full".equals(data.getString("snapshot_type"))) {
                baseContent = data.getJSONObject("content");
                break;
            }
        }
        
        if (baseContent == null) {
            throw new BusinessException("找不到基础全量快照");
        }
        
        // 3. 按顺序应用增量变更
        JSONObject result = JSONObject.parseObject(baseContent.toJSONString()); // 深拷贝
        
        for (DraftAutoSave snapshot : snapshotChain) {
            JSONObject data = JSONObject.parseObject(snapshot.getSaveData());
            if ("incremental".equals(data.getString("snapshot_type"))) {
                applyIncrementalChanges(result, data);
            }
        }
        
        return result;
    }
    
    /**
     * 构建快照链（从目标快照回溯到基础快照）
     */
    private List<DraftAutoSave> buildSnapshotChain(DraftAutoSave targetSnapshot) {
        List<DraftAutoSave> chain = new ArrayList<>();
        DraftAutoSave current = targetSnapshot;
        
        while (current != null) {
            chain.add(0, current); // 插入到开头，保持时间顺序
            
            JSONObject data = JSONObject.parseObject(current.getSaveData());
            if ("full".equals(data.getString("snapshot_type"))) {
                break; // 找到基础全量快照
            }
            
            // 查找父快照
            String parentId = current.getParentSaveId();
            if (parentId != null) {
                current = autoSaveMapper.selectById(parentId);
            } else {
                current = null;
            }
        }
        
        return chain;
    }
    
    /**
     * 应用增量变更到内容对象
     */
    private void applyIncrementalChanges(JSONObject content, JSONObject incrementalData) {
        JSONArray operations = incrementalData.getJSONObject("changes").getJSONArray("operations");
        
        for (int i = 0; i < operations.size(); i++) {
            JSONObject operation = operations.getJSONObject(i);
            String type = operation.getString("type");
            String path = operation.getString("path");
            
            switch (type) {
                case "update":
                    setValueByPath(content, path, operation.get("new_value"));
                    break;
                case "insert":
                    setValueByPath(content, path, operation.get("value"));
                    break;
                case "delete":
                    removeValueByPath(content, path);
                    break;
            }
        }
    }
    
    /**
     * 根据路径设置值
     */
    private void setValueByPath(JSONObject obj, String path, Object value) {
        String[] parts = path.split("\\.");
        JSONObject current = obj;
        
        for (int i = 0; i < parts.length - 1; i++) {
            String part = parts[i];
            if (!current.containsKey(part)) {
                current.put(part, new JSONObject());
            }
            current = current.getJSONObject(part);
        }
        
        current.put(parts[parts.length - 1], value);
    }
    
    /**
     * 根据路径删除值
     */
    private void removeValueByPath(JSONObject obj, String path) {
        String[] parts = path.split("\\.");
        JSONObject current = obj;
        
        for (int i = 0; i < parts.length - 1; i++) {
            String part = parts[i];
            if (!current.containsKey(part)) {
                return; // 路径不存在
            }
            current = current.getJSONObject(part);
        }
        
        current.remove(parts[parts.length - 1]);
    }
}
```

```java
/**
 * 数据压缩工具
 * 对快照数据进行压缩以减少存储空间
 */
@Component
public class SnapshotCompressor {
    
    /**
     * 压缩快照数据
     * @param data 原始数据
     * @return 压缩后的数据
     */
    public byte[] compress(String data) {
        try {
            ByteArrayOutputStream baos = new ByteArrayOutputStream();
            try (GZIPOutputStream gzipOut = new GZIPOutputStream(baos)) {
                gzipOut.write(data.getBytes(StandardCharsets.UTF_8));
            }
            return baos.toByteArray();
        } catch (IOException e) {
            throw new RuntimeException("数据压缩失败", e);
        }
    }
    
    /**
     * 解压快照数据
     * @param compressedData 压缩数据
     * @return 原始数据
     */
    public String decompress(byte[] compressedData) {
        try {
            ByteArrayInputStream bais = new ByteArrayInputStream(compressedData);
            try (GZIPInputStream gzipIn = new GZIPInputStream(bais)) {
                return new String(gzipIn.readAllBytes(), StandardCharsets.UTF_8);
            }
        } catch (IOException e) {
            throw new RuntimeException("数据解压失败", e);
        }
    }
    
    /**
     * 计算压缩比
     * @param originalSize 原始大小
     * @param compressedSize 压缩后大小
     * @return 压缩比
     */
    public double calculateCompressionRatio(int originalSize, int compressedSize) {
        if (originalSize == 0) {
            return 0.0;
        }
        return (double) compressedSize / originalSize;
    }
}
```

### 7.2 内容哈希去重

```java
/**
 * 内容哈希管理器
 * 通过哈希值实现内容去重
 */
@Service
public class ContentHashManager {
    
    @Autowired
    private DraftAutoSaveMapper autoSaveMapper;
    
    /**
     * 计算内容哈希
     * @param content 内容对象
     * @return SHA-256哈希值
     */
    public String calculateContentHash(JSONObject content) {
        try {
            // 标准化内容（排序键值对）
            String normalizedContent = normalizeContent(content);
            
            MessageDigest digest = MessageDigest.getInstance("SHA-256");
            byte[] hash = digest.digest(normalizedContent.getBytes(StandardCharsets.UTF_8));
            
            return bytesToHex(hash);
        } catch (NoSuchAlgorithmException e) {
            throw new RuntimeException("哈希算法不支持", e);
        }
    }
    
    /**
     * 检查是否存在相同内容的快照
     * @param draftId 草稿ID
     * @param contentHash 内容哈希
     * @return 存在的快照ID，不存在返回null
     */
    public String findExistingSnapshot(String draftId, String contentHash) {
        QueryWrapper<DraftAutoSave> queryWrapper = new QueryWrapper<>();
        queryWrapper.eq("draft_id", draftId)
                   .eq("content_hash", contentHash)
                   .orderByDesc("create_time")
                   .last("LIMIT 1");
        
        DraftAutoSave existing = autoSaveMapper.selectOne(queryWrapper);
        return existing != null ? existing.getId() : null;
    }
    
    /**
     * 标准化内容（确保相同内容产生相同哈希）
     */
    private String normalizeContent(JSONObject content) {
        // 创建TreeMap确保键的排序一致性
        Map<String, Object> sortedMap = new TreeMap<>();
        content.forEach(sortedMap::put);
        
        return JSONObject.toJSONString(sortedMap, SerializerFeature.SortField);
    }
    
    /**
     * 字节数组转十六进制字符串
     */
    private String bytesToHex(byte[] bytes) {
        StringBuilder result = new StringBuilder();
        for (byte b : bytes) {
            result.append(String.format("%02x", b));
        }
        return result.toString();
    }
}
```

```java
/**
 * 异步保存服务
 * 使用异步机制提高保存性能
 */
@Service
public class AsyncSaveService {
    
    private static final Logger logger = LoggerFactory.getLogger(AsyncSaveService.class);
    
    @Autowired
    private DraftAutoSaveService autoSaveService;
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    // 异步执行器
    private final Executor saveExecutor = Executors.newFixedThreadPool(5, 
        new ThreadFactoryBuilder().setNameFormat("draft-save-%d").build());
    
    /**
     * 异步保存草稿
     * @param draftId 草稿ID
     * @param content 内容
     * @param saveType 保存类型
     */
    @Async
    public CompletableFuture<String> saveAsync(String draftId, JSONObject content, Integer saveType) {
        return CompletableFuture.supplyAsync(() -> {
            try {
                long startTime = System.currentTimeMillis();
                
                // 执行保存
                String saveId = autoSaveService.saveSnapshot(draftId, content, saveType);
                
                long duration = System.currentTimeMillis() - startTime;
                logger.info("草稿异步保存完成，draftId: {}, saveId: {}, 耗时: {}ms", 
                    draftId, saveId, duration);
                
                // 更新缓存中的最新保存时间
                updateLastSaveTime(draftId);
                
                return saveId;
            } catch (Exception e) {
                logger.error("草稿异步保存失败，draftId: {}", draftId, e);
                throw new RuntimeException("保存失败", e);
            }
        }, saveExecutor);
    }
    
    /**
     * 批量异步保存
     * @param saveRequests 保存请求列表
     */
    public CompletableFuture<List<String>> batchSaveAsync(List<SaveRequest> saveRequests) {
        List<CompletableFuture<String>> futures = saveRequests.stream()
            .map(request -> saveAsync(request.getDraftId(), request.getContent(), request.getSaveType()))
            .collect(Collectors.toList());
        
        return CompletableFuture.allOf(futures.toArray(new CompletableFuture[0]))
            .thenApply(v -> futures.stream()
                .map(CompletableFuture::join)
                .collect(Collectors.toList()));
    }
    
    /**
     * 更新最后保存时间缓存
     */
    private void updateLastSaveTime(String draftId) {
        String cacheKey = "draft:last_save_time:" + draftId;
        redisTemplate.opsForValue().set(cacheKey, System.currentTimeMillis(), Duration.ofHours(24));
    }
    
    /**
     * 保存请求对象
     */
    @Data
    public static class SaveRequest {
        private String draftId;
        private JSONObject content;
        private Integer saveType;
    }
}
```

```java
     * 清理过期的临时快照
     */
    private int cleanupExpiredTemporarySnapshots() {
        QueryWrapper<DraftAutoSave> queryWrapper = new QueryWrapper<>();
        queryWrapper.lt("expire_time", new Date())
                   .eq("is_permanent", 0);
        
        return autoSaveMapper.delete(queryWrapper);
    }
    
    /**
     * 清理冗余的自动保存快照
     * 每个草稿保留最近50个自动保存快照
     */
    private int cleanupRedundantAutoSaves() {
        // 获取所有有自动保存记录的草稿
        List<String> draftIds = autoSaveMapper.selectDistinctDraftIds();
        
        int totalCleaned = 0;
        
        for (String draftId : draftIds) {
            // 获取该草稿的所有自动保存记录，按时间倒序
            QueryWrapper<DraftAutoSave> queryWrapper = new QueryWrapper<>();
            queryWrapper.eq("draft_id", draftId)
                       .eq("save_type", 1) // 自动保存
                       .eq("is_permanent", 0) // 非永久保存
                       .orderByDesc("create_time");
            
            List<DraftAutoSave> autoSaves = autoSaveMapper.selectList(queryWrapper);
            
            // 保留最近50个，删除其余的
            if (autoSaves.size() > 50) {
                List<String> toDeleteIds = autoSaves.stream()
                    .skip(50)
                    .map(DraftAutoSave::getId)
                    .collect(Collectors.toList());
                
                if (!toDeleteIds.isEmpty()) {
                    int deleted = autoSaveMapper.deleteBatchIds(toDeleteIds);
                    totalCleaned += deleted;
                    logger.debug("草稿 {} 清理了 {} 个冗余快照", draftId, deleted);
                }
            }
        }
        
        return totalCleaned;
    }
    
    /**
     * 压缩历史快照
     * 将7天前的快照进行压缩存储
     */
    private int compressHistoricalSnapshots() {
        Date sevenDaysAgo = Date.from(Instant.now().minus(7, ChronoUnit.DAYS));
        
        QueryWrapper<DraftAutoSave> queryWrapper = new QueryWrapper<>();
        queryWrapper.lt("create_time", sevenDaysAgo)
                   .isNull("compression_ratio"); // 未压缩的
        
        List<DraftAutoSave> uncompressedSnapshots = autoSaveMapper.selectList(queryWrapper);
        
        int compressedCount = 0;
        SnapshotCompressor compressor = new SnapshotCompressor();
        
        for (DraftAutoSave snapshot : uncompressedSnapshots) {
            try {
                String originalData = snapshot.getSaveData();
                byte[] compressedData = compressor.compress(originalData);
                
                // 更新压缩数据和压缩比
                snapshot.setSaveData(Base64.getEncoder().encodeToString(compressedData));
                snapshot.setCompressionRatio(
                    compressor.calculateCompressionRatio(originalData.length(), compressedData.length)
                );
                
                autoSaveMapper.updateById(snapshot);
                compressedCount++;
                
            } catch (Exception e) {
                logger.warn("快照压缩失败，ID: {}", snapshot.getId(), e);
            }
        }
        
        return compressedCount;
    }
}
```

## 11. 总结

本设计方案通过以下关键特性实现了高效可靠的草稿自动保存功能：

1. **分层快照策略**：L1/L2/L3三层快照，平衡性能和存储

6. **自动清理策略**：控制存储成本，维护系统性能
7. **完善监控体系**：实时监控系统状态，及时发现问题

该方案既保证了数据安全性，又优化了性能和存储效率，为用户提供了可靠的内容编辑体验。
